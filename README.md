</p>

## 📖 Overview

**Sofra** is a modern restaurant management system designed to simplify restaurant operations through an intuitive role-based dashboard.

The system provides complete management for:

- 🍽 Menu Management
- 📦 Inventory Management
- 📋 Orders
- 👨‍🍳 Kitchen Workflow
- 💳 Billing & Payments
- 📊 Reports & Analytics
- 🎁 Offers & Discounts
- 📱 QR Digital Menu
- 👥 User Roles & Permissions

The project was developed as a graduation project while following Laravel best practices.

# 🚀 Installation

```bash
git clone https://github.com/Saleh-Wolf/sofra.git

cd sofra

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate --seed

npm install

npm run build

php artisan serve
```
